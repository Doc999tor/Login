# Customers

### Usage
```
npm install
npm start
Open http://localhost:3000
```
### Build & Watch
Building  bundle.js and bundle.css to directory dist.
```
npm run build
```
```
npm run watch
Open http://localhost:8080
```
### Build for production
```
npm run build-p
```

### e2e-testing
  ```
    npm install -g webdriver-manager
    npm install -g protractor
    webdriver-manager update
  ```
### Start test
  ```
    npm start
    npm run webdriver
    npm run test
  ```
